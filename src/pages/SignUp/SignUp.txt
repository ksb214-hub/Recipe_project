SignUpPage 설계 문서
 
1. 개요(Overview)
    SignUpPage는 회원가입을 할수 있도록 하는 페이지이다.
    사용자는 아이디, 비번, 이름, 이메일, 생년월일, 성별, 휴대폰 번호, 인증을 입력하여 
    자신의 개인정보를 입력할 수 있게 한다.
    
    등록된 개인정보를 통해 해당 사용자는 해당 서비스에 로그인을 실행할수 있도록 한다.

2. 개발 환경
    항목		내용
	Framework	React
	Language	JavaScript
	Routing		React Router
	Component	Input, Button
	Styling	    CSS

3. 폴더 구조
    plaintext
        src/
        │
        ├── pages/
        │   └── SignUp/
        │       ├── SignUpPage.js
        │       └── SignUpPage.css

    구성 요소
        SignUpPage.js
        → 사용자 개인정보 등록 및 관리 기능 구현

        SignUpPage.css
        → 회원가입 페이지 UI 스타일

4. 목적
    SignUpPage는 다음 기능을 제공한다.
        개인정보 등록
        아이디 입력
        비번 입력      

    이를 통해 사용자는 자신의 개인정보를 등록 하고 로그인 할 수 있다.
